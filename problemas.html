Problemas com o Arcade?

Soluções rápidas para questões técnicas

Problemas Frequentes

Não consigo logar com meu RM

  1. Verifique se digitou seu RM corretamente
  2. Confirme com a coordenação se seu cadastro está ativo
  3. Se o problema persistir, reinicie o sistema do arcade (botão lateral)

Os controles não estão respondendo

  1. Verifique se o jogo já foi iniciado (crédito debitado)
  2. Conferir se todos os cabos estão firmemente conectados
  3. Limpe os botões com pano seco (sem produtos químicos)

Tela congelada ou sem imagem

  1. Reinicie o arcade usando o botão de energia
  2. Verifique se o monitor está recebendo energia
  3. Confira a conexão do cabo VGA/HDMI na parte traseira

Créditos não aparecem após doação

  1. Verifique com a coordenação se sua doação já foi registrada
  2. Espere até 24h para atualização do sistema
  3. Caso persista, solicite ajuda técnica na secretaria
